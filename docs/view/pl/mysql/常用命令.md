# 常用命令

#### 登录

```sql
mysql -u username -p password
```

#### 显示数据库列表

```
SHOW DATABASES;
```

#### 使用数据库

```sql
USE 'name';
```

#### 显示表列表

```sqsl
SHOW TABLES;
```

#### 显示表结构

```sql
DESC table_name;
```

#### 查询

```sql
SELECT
```

#### 增加

```sql
insert
```

#### 删除

```sql
delete
```

#### 更新

```sql
UPDATE 
```
